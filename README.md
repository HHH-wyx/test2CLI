# 出生日期查询器

一个简洁实用的在线工具，用于计算从出生日期到任意目标日期的天数差。界面采用清新的绿白色配色，操作简单直观。

## 🌟 功能特性

- **精确计算**：准确计算两个日期之间的天数差
- **简洁界面**：绿白色清爽设计，现代感十足
- **快速选择**：内置常用日期快速选择功能
- **动画效果**：数字动画展示计算结果
- **响应式设计**：完美适配各种设备屏幕
- **输入验证**：智能校验日期输入合法性

## 🚀 快速开始

### 在线使用

直接打开 `index.html` 文件即可在浏览器中使用，无需任何安装。

### 使用步骤

1. **输入出生日期**：在第一个输入框选择您的出生日期
2. **选择目标日期**：在第二个输入框选择要计算的目标日期
3. **点击计算**：点击"计算天数"按钮或按回车键
4. **查看结果**：页面将显示精确的天数计算结果

## 🎯 快速选择功能

为方便使用，内置了快速选择按钮：

- **今天**：计算到当前日期的天数
- **本周**：计算到本周最后一天的天数  
- **本月**：计算到本月最后一天的天数
- **今年**：计算到今年最后一天的天数

## 💻 技术实现

### 核心算法

```javascript
function calculateDaysBetween(startDate, endDate) {
    const start = new Date(startDate);
    const end = new Date(endDate);
    
    // 清除时间部分，确保只比较日期
    start.setHours(0, 0, 0, 0);
    end.setHours(0, 0, 0, 0);
    
    const timeDiff = end.getTime() - start.getTime();
    return Math.floor(timeDiff / (1000 * 60 * 60 * 24));
}
```

## 🆘 常见问题

### Q: 为什么计算结果不准确？
A: 请确保输入的日期格式正确，且出生日期早于目标日期。

### Q: 能否计算未来日期？
A: 可以，系统支持计算任意两个日期之间的天数差。

### Q: 是否支持农历日期？
A: 目前仅支持公历日期计算。
